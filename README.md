## 🧪 Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/dhomini-pereira/portfolio.git

# Acesse a pasta
cd portfolio

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```
